Core Commands

終端機裡
的開發團隊

六個核心命令，管理角色、任務、工具。不需要 UI，不需要點擊 —— 一個命令就是一個動作。支援 Claude Code 插件模式，也可以獨立啟動 MCP Server。

$ bun run dev -- roles 列出 18 個角色
$ bun run dev -- role qa-engineer 切換當前角色
$ bun run dev -- task "測試登入" 派發任務
$ bun run dev -- list 查看任務看板
$ bun run dev -- tools 列出 7 個工具
$ bun run dev -- tool web_search 運行工具
─────── $ bun run desktop-server 桌面 MCP
$ bun run core-tools-server 工具 MCP
